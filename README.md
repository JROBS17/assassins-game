# Assassin Tag Game

## Setup Instructions

1. Create a virtual environment:
   ```
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the app:
   ```
   flask run
   ```

4. Visit `http://127.0.0.1:5000` in your browser.
